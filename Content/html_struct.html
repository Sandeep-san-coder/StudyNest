Structure of HTML

ðŸ”¹ What is HTML?

HTML is a markup language used for structuring content on the web. It consists of **tags** enclosed in angle brackets like <html>.

ðŸ”¹ Basic Structure of an HTML Page

A simple HTML document includes:

  • <!DOCTYPE html> - Declares the document type
  • <html> - Root element
  • <head> - Contains metadata (title, styles)
  • <body> - Main content of the webpage

ðŸ”¹ Example Code:

<!DOCTYPE html>
<html>
    <head>
        <title>My First Page</title>
    </head>
    <body>
        <h1>Hello, World!</h1>
        <p>This is my first webpage.</p>
    </body>
</html>
