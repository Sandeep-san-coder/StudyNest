Fundamentals of Computers - Unit 2

Computer Software

Software is a generic term for recognized collection of computer data and instructions. It is responsible for controlling, integrating, and managing the hardware components of a computer and to accomplish specific tasks.

Relationship between Software and Hardware

Software refers to the computer programs that are loaded into a computer system.

Hardware refers to all visible devices, which are assembled to build a computer system.

Categories of Software

System Software

System software is the software designed to control and manage the hardware of a computer.

Examples:

  • Operating Systems (Windows, Linux, macOS, Android)
  • Utility Programs (antivirus, file management tools, disk cleaners)
  • Device Drivers (printer driver, graphics driver)
  • Firmware (software embedded in hardware devices)

Operating System

An Operating System (OS) is a type of system software that acts as an interface between the computer hardware and the user.

Process Management

Handles the creation, deletion, suspension, resumption, and synchronization of processes.

Memory Management

Handles allocation and de-allocation of memory space as required by various programs.

File Management

Responsible for the creation and deletion of files and directories.

Device Management

Manages all the devices of the computer system such as printers and modems.

Security Management

Protects the system resources and information against destruction and unauthorized use.

User Interface

Provides the interface between the user and the hardware.

Device Drivers

A Device Driver is a specialized program that allows the operating system and application software to communicate with hardware devices.
